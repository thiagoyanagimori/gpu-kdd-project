# GPU Price Prediction

Projeto de análise de dados utilizando Python e Machine Learning para prever preços de GPUs com base em benchmark e TDP.

## Tecnologias
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## Modelos utilizados
- Regressão Linear
- Árvore de Decisão

## Objetivo
Analisar a relação entre desempenho de GPUs e seus preços.
